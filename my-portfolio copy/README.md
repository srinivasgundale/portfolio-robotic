# Full Stack Developer Portfolio

Built with Next.js, Tailwind CSS, GSAP, Three.js, and AI.

## Features

- **3D Background**: Interactive star field using React Three Fiber.
- **GSAP Animations**: Scroll-triggered animations for sections.
- **Chatbot**: AI-powered assistant (mocked for demo) that answers questions about the portfolio.
- **Responsive Design**: Fully responsive layout with a dark premium theme.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- Next.js 16
- Tailwind CSS v4
- GSAP
- Three.js / React Three Fiber
- Framer Motion
- Lucide React
